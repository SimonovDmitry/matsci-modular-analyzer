# Автоматизированный анализ микроструктур руды

> **End-to-end пайплайн для объективной оценки технологических свойств сырья**
> Наше решение объединяет ансамбль из двух нейросетей для точной классификации типов срастаний и попиксельной сегментации талька. Архитектура построена на базе асинхронного FastAPI-сервера с интуитивно понятным интерактивным дашбордом на Streamlit, что делает сложный ML-инструмент доступным для профильных специалистов.

Продукт эффективно решает проблему субъективности визуальной экспертизы, обрабатывая тяжелые панорамные снимки за считанные минуты, и автоматически генерирует строгие PDF-отчеты академического уровня, готовые к интеграции в исследовательские процессы НИИ.

---

## Проблематика

В геологоразведке и обогащении руд оперативная оценка свойств сырья критически важна. Традиционный ручной анализ микрофотографий полированных шлифов имеет ряд ограничений:
* **Субъективность:** Разные специалисты по-разному оценивают долю фаз.
* **Трудоемкость:** Ручная сегментация гигапиксельных изображений занимает часы на один образец.
* **Масштабируемость:** Анализ партий шлифов становится «бутылочным горлышком» лаборатории.
* **Вариативность данных:** Освещение, артефакты полировки и царапины сбивают жесткие алгоритмы.

---

## Архитектура и стек технологий

* **Backend:** FastAPI (асинхронная обработка тяжелых изображений).
* **Frontend:** Streamlit (интерактивный дашборд для геологов).
* **ML Core:** PyTorch (ансамбль из двух нейросетей).
* **Computer Vision:** OpenCV, scikit-image (предобработка нормализация освещения, работа с TIFF/PNG).
* **Экспорт отчетов:** Генерация PDF-документов с метриками и визуализацией.

---

## Логика классификации и сегментации

Пайплайн выполняет попиксельную сегментацию, выделяя ключевые фазы (цветовая маска):
* **Синий:** Тальк (темная рассеянная фаза).

Решение использует строгую экспертную логику для итоговой классификации руды:

| Доля талька | Преобладающий тип срастаний | Итоговый класс руды |
| :--- | :--- | :--- |
| **> 10%** | Любые | **Оталькованная руда** |
| **≤ 10%** | Обычные срастания | **Рядовая руда** |
| **≤ 10%** | Тонкие срастания | **Труднообогатимая руда** |

## Ключевые возможности

### Функциональные
* Поддержка форматов высокого разрешения (TIFF, PNG, JPEG).
* Автоматическая нормализация освещения, шумоподавление и коррекция контраста.
* Расчет площадей и процентных долей фаз с учетом масштаба.
* Интерактивный просмотр результатов с возможностью зума.
* Пакетная обработка серий изображений без участия пользователя.
* Экспорт количественных метрик в CSV и генерация итогового PDF-отчета.

### Нефункциональные (Метрики)
* **Производительность:** Обработка панорамного снимка (до 10 000 × 10 000 пикселей) занимает менее 5 минут.
* **Точность (Тальк):** Ошибка оценки доли талька не превышает ±3% относительно разметки.
* **Точность (Срастания):** F1-score классификации типов срастаний > 75%.
* **Надежность:** Устойчивость к царапинам, загрязнениям шлифа и неравномерному освещению.
* **Безопасность:** Возможность полностью локального развертывания (On-Premise) для конфиденциальных данных.

## Системные требования
 * Python: 3.10 – 3.13 (тестировалось на macOS Apple Silicon M1 и Windows 11)
 * Зависимости: Весь необходимый стек (FastAPI, Streamlit, PyTorch, Albumentations) зафиксирован в файле requirements.txt.

## Инструкция по локальному запуску
### 1. Клонирование репозитория и переход в папку проектаgit clone <ссылка на репозиторий>
cd matsci-modular-analyzer

### 2. Развертывание виртуального окружения и установка зависимостей
# Создание виртуального окружения
python3 -m venv venv

Активация для macOS/Linux: source venv/bin/activate
Или активация для Windows (PowerShell): .\venv\Scripts\Activate.ps1

# Установка всех необходимых библиотек
pip install --upgrade pip
pip install -r requirements.txt

### 3. Запуск Бэкенда (FastAPI)
Сервер автоматической обработки изображений и инференса нейросетей должен быть запущен из директории api:cd api
uvicorn main:app --host 127.0.0.1 --port 8000

*Интерактивная документация API Swagger будет доступна по адресу:* http://127.0.0.1:8000/docs
### 4. Запуск Фронтенда / Дашборда (Streamlit)
Откройте новое окно терминала, активируйте виртуальное окружение (source venv/bin/activate) и запустите интерфейс из корневой директории проекта:streamlit run app.py

*Веб-интерфейс автоматически откроется в браузере по адресу:* http://localhost:8501
